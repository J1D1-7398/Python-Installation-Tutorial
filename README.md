# Python-Installation
A simple tutorial on how to install Python on a Windows system. I will update this repo regularly when the installation process changes.

## **How to install python**

Go to the official website of python (https://www.python.org/)

On the menu, press "Downloads"

<img width="1161" height="572" alt="Pythondownload" src="https://github.com/user-attachments/assets/4fc94492-cd18-45ad-9a39-eaeccda6c096" />

You will see "Or get the standalone installer for Python 3.x.x" under "Python install manager"

Click the link "Python 3.x.x"

Download will start

<img width="522" height="177" alt="Pythondownload1" src="https://github.com/user-attachments/assets/88a46d65-a358-4497-a0fb-c2756df33a4d" />

When the download finishes, go to downloads (Or the folder you previously chose to download files) and double click it

Go to custom installation to specify custom features (I recommend to check the option "tcl/tk and IDLE" because some of my projects use tkinter (tk))

Recommendations (all these options):

<img width="621" height="361" alt="Screenshot 2025-11-26 190537" src="https://github.com/user-attachments/assets/34836913-f95e-458a-bc27-0e10d0e6a570" />

Follow the instructions on screen

Finish the installation process by clicking "Install"
